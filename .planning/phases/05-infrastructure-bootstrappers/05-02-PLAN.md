    Create directory `src/Cache/` and file `src/Cache/TenantAwareCacheAdapter.php`:

    ```php
    <?php
    declare(strict_types=1);
    namespace Tenancy\Bundle\Cache;

    use Psr\Cache\CacheItemInterface;
    use Symfony\Component\Cache\Adapter\AdapterInterface;
    use Symfony\Component\Cache\CacheItem;
    use Symfony\Contracts\Cache\NamespacedPoolInterface;
    use Tenancy\Bundle\Context\TenantContext;

    final class TenantAwareCacheAdapter implements AdapterInterface, NamespacedPoolInterface
    {
        public function __construct(
            private AdapterInterface&NamespacedPoolInterface $inner,
            private readonly TenantContext $tenantContext,
        ) {
        }

        private function pool(): AdapterInterface&NamespacedPoolInterface
        {
            if ($this->tenantContext->hasTenant()) {
                return $this->inner->withSubNamespace(
                    $this->tenantContext->getTenant()->getSlug()
                );
            }
            return $this->inner;
        }

        public function getItem(mixed $key): CacheItem
        {
            return $this->pool()->getItem($key);
        }

        /** @return iterable<string, CacheItem> */
        public function getItems(array $keys = []): iterable
        {
            return $this->pool()->getItems($keys);
        }

        public function hasItem(mixed $key): bool
        {
            return $this->pool()->hasItem($key);
        }

        public function clear(string $prefix = ''): bool
        {
            return $this->pool()->clear($prefix);
        }

        public function deleteItem(mixed $key): bool
        {
            return $this->pool()->deleteItem($key);
        }

        public function deleteItems(array $keys): bool
        {
            return $this->pool()->deleteItems($keys);
        }

        public function save(CacheItemInterface $item): bool
        {
            return $this->pool()->save($item);
        }

        public function saveDeferred(CacheItemInterface $item): bool
        {
            return $this->pool()->saveDeferred($item);
        }

        public function commit(): bool
        {
            return $this->pool()->commit();
        }

        public function withSubNamespace(string $namespace): static
        {
            $clone = clone $this;
            $clone->inner = $this->inner->withSubNamespace($namespace);
            return $clone;
        }
    }
    ```

    Note: The `$inner` property is NOT readonly because `withSubNamespace()` needs to modify it on the clone. The `private` visibility plus `final class` prevents external mutation.

    Note: The `pool()` method is called on every operation to read TenantContext live. This is the critical design — never cache the result of withSubNamespace() across calls.

    2. Add service definition after the existing services (before the closing `};`):
    ```php
    $services->set('tenancy.cache_adapter', TenantAwareCacheAdapter::class)
        ->decorate('cache.app')
        ->args([
            service('.inner'),
            service('tenancy.context'),
        ]);
    ```

    The `->decorate('cache.app')` tells Symfony to:
    - Rename the original `cache.app` service to `tenancy.cache_adapter.inner`
    - Register `TenantAwareCacheAdapter` as the new `cache.app`
    - `service('.inner')` is the Symfony convention for referencing the decorated (original) service

    This is always-on — no conditional wiring, no config flags. Any service injecting `cache.app` (including `DoctrineTenantProvider`) gets the tenant-aware adapter transparently.
